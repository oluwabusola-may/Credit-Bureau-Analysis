1. How many Customers are in the dataset?

SQL QUERY:
SELECT COUNT(*) AS total_customers FROM credits;

Insights: There are 10,000 total customers in the dataset.


2. What is the default rate across the portfolio?

SQL QUERY:
SELECT Default_Status, 
COUNT(*) AS count, 
ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER (), 2)
AS percentage FROM credits
GROUP BY Default_Status;

Insights: 33% of customers have defaulted. This is very high, a healthy portfolio sits below 10%. The entire portfolio needs tighter credit risk controls.


3. How many customers fall into each risk category?

SQL QUERY:
SELECT Risk_Category, 
COUNT(*) AS customer_count 
FROM credits 
GROUP BY Risk_Category
ORDER BY customer_count 
DESC;

Insights:67% High Risk, 31% Medium, only 1% Low. The portfolio is heavily skewed toward risky customers. The bank is likely approving too many borderline applicants.


4. What is the average credit score by loan type?

SQL QUERY:
SELECT Loan_Type, 
ROUND(AVG(Credit_Score), 1) AS avg_credit_score 
FROM credits 
WHERE
Credit_Score IS NOT NULL
 GROUP BY Loan_Type 
 ORDER BY avg_credit_score 
 DESC;

Insights: All loan types cluster tightly between 605–607. No loan type is attracting significantly better-quality borrowers, which explains the uniformly high 
default rate across the board.


5. Which states have a default rate above the national average?

SQL QUERY:
WITH national_avg AS
 ( SELECT AVG(CASE WHEN Default_Status = 'Yes' THEN 1.0 ELSE 0 END) AS
nat_default_rate FROM credits),
 state_rates AS ( SELECT State, AVG(CASE WHEN
Default_Status = 'Yes' THEN 1.0 ELSE 0 END) AS state_default_rate,
 COUNT(*) AS total_customers
FROM credits
 GROUP BY State)
 SELECT s.State,
 ROUND(s.state_default_rate * 100, 2) AS default_rate_pct,
 ROUND(n.nat_default_rate * 100, 2) AS national_avg_pct
 FROM state_rates s
CROSS JOIN national_avg n
 WHERE s.state_default_rate > n.nat_default_rate
 ORDER BY
s.state_default_rate
 DESC;

Insights:Rivers (63%), Kano (34%) and Delta (34%) all exceed the 33% national average. Rivers is a major red flag, its default rate is nearly double the average. 
Consider stricter lending criteria or reduced exposure in that region.


6. Delinquency breakdown by loan type 

SQL QUERY:
SELECT Loan_Type, 
Delinquency_Status, 
COUNT(*) AS count, 
ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) 
OVER (PARTITION BY Loan_Type), 2) AS pct_within_loan_type 
FROM credits
GROUP BY Loan_Type, Delinquency_Status 
ORDER BY Loan_Type, 
count DESC;

Insights: Payday loans are the most dangerous, only 48% of customers are current, meaning over half are already late. Personal loans follow with 32% delinquency. 
Mortgage loans are the healthiest at 88% current. Payday lending should be reviewed urgently.


7. High-utilization, high-DTI customers who haven't yet defaulted

SQL QUERY:
SELECT Customer_ID, Credit_Score, 
Credit_Utilization_Rate, Debt_to_Income_Ratio,
Outstanding_Balance
 FROM credits
 WHERE Credit_Utilization_Rate > 0.70 
 AND Debt_to_Income_Ratio > 0.50 
 AND Default_Status = 'No' 
 ORDER BY Outstanding_Balance 
 DESC 
 LIMIT 50;

Insights: These customers have used over 70% of their credit and are spending more than 50% of income on debt, but haven't defaulted yet. They are pre-default cases
 and represent the highest immediate risk to the portfolio.


8. Year-over-year credit score trend for High-Risk customers

SQL QUERY:
SELECT Report_Year, 
ROUND(AVG(CASE WHEN Risk_Category = 'High' THEN Credit_Score END), 1) AS avg_score_high_risk, 
ROUND(AVG(CASE WHEN Risk_Category = 'Medium' THEN Credit_Score END), 1) AS avg_score_medium_risk, 
ROUND(AVG(CASE WHEN Risk_Category = 'Low' THEN Credit_Score END), 1) AS avg_score_low_risk 
FROM credits 
WHERE Credit_Score IS NOT NULL 
GROUP BY Report_Year
ORDER BY Report_Year;

Insights:Scores are essentially flat across 2022–2024 (High Risk ~591, Medium ~632, Low ~711). No meaningful improvement or deterioration meaning the portfolio quality 
is stagnant, not recovering.


9. Rank states by total outstanding exposure using window functions

SQL QUERY:
SELECT State, COUNT(*) AS customers, 
ROUND(SUM(Outstanding_Balance), 2) AS total_exposure,
ROUND(AVG(Outstanding_Balance), 2) AS avg_exposure, 
RANK() OVER 
(ORDER BY
SUM(Outstanding_Balance) DESC) AS exposure_rank 
FROM credits
 GROUP BY State
 ORDER BY exposure_rank;

Insights: Lagos carries the highest total exposure (₦117B) due to customer volume, followed by Abuja (₦75B) and Rivers (₦54B). Rivers is particularly concerning because its 
exposure is high and its default rate is the worst which is a dangerous combination.

10. Top 10% most at-risk customers via composite risk score

SQL QUERY:
WITH risk_scored AS 
( SELECT 
Customer_ID, Credit_Score, Debt_to_Income_Ratio,
Credit_Utilization_Rate, Number_of_Late_Payments, Outstanding_Balance,
 (850 -
COALESCE(Credit_Score, 500)) * 0.4 + COALESCE(Debt_to_Income_Ratio, 0) * 100 * 0.3 +
COALESCE(Credit_Utilization_Rate, 0) * 100 * 0.2 + Number_of_Late_Payments * 5 * 0.1 
AS composite_risk_score FROM credits), 
percentiled AS 
( SELECT *, NTILE(10) OVER 
(ORDER BY composite_risk_score DESC) 
AS decile FROM risk_scored ) 
SELECT Customer_ID, Credit_Score, Debt_to_Income_Ratio, Credit_Utilization_Rate, composite_risk_score 
FROM percentiled 
WHERE decile = 1 
ORDER BY composite_risk_score DESC;

Insights: The riskiest customers all have DTI maxed at 1.5 and credit scores mostly below 480. These customers should be flagged for immediate account review, reduced credit limits
or collections outreach.

11. Loan type risk profiling, default rate and exposure ranked

SQL QUERY:
WITH loan_summary AS 
( SELECT Loan_Type, COUNT(*) AS total_customers, 
ROUND(AVG(Loan_Amount), 2) AS avg_loan_amount, 
ROUND(AVG(Outstanding_Balance), 2) AS avg_outstanding,
ROUND(AVG(Number_of_Late_Payments), 2) AS avg_late_payments, 
ROUND(AVG(CASE WHEN Default_Status = 'Yes' THEN 1.0 ELSE 0 END) * 100, 2) AS default_rate_pct,
ROUND(SUM(Outstanding_Balance), 2) AS total_exposure
FROM credits 
GROUP BY Loan_Type )
SELECT *, 
RANK() OVER (ORDER BY default_rate_pct DESC) AS default_rank, 
RANK() OVER (ORDER BY total_exposure DESC) AS exposure_rank 
FROM loan_summary 
ORDER BY default_rate_pct DESC;

Insights:Mortgage has the highest default rate (38%) and the largest total exposure (₦311B) — this is the single biggest risk in the portfolio. Despite appearing "safe" on delinquency, 
mortgage defaults are catastrophically expensive when they do occur. Business loans rank second on both metrics.
